# Security Policy

## Reporting a vulnerability

Please **do not open a public issue** for security vulnerabilities.

Email **derrynknife@gmail.com** with a description of the issue, steps to
reproduce, and the affected version/commit. You'll get an acknowledgement
within a few days, and a fix will be prioritised.

## Scope

- The web application (backend and frontend) in this repository.
- Self-hosted deployments run in single-user mode by default; operators are
  responsible for not exposing a single-user instance to untrusted networks.
